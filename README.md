# Bloomberg Terminal Dashboard

Financial information dashboard with real-time data visualization.

## Features

- World clocks and global indices
- Commodities, crypto, FX, bonds
- Sector heatmap
- News ticker
- Flight tracker

## Tech Stack

- Pure HTML/CSS/JavaScript
- Python simple HTTP server with basic auth
- Data fetched via Python scripts

## Setup

```bash
# Run dashboard
sudo python3 server.py

# Access at http://localhost
```

## Data Sources

- AI News
- GitHub Trending
- Geopolitics RSS
- Market Indices
